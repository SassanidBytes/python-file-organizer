# Einfaches Projekt damit ordner inhalte ordentlich sortieren kann
Es sortiert Dateien in einem Ordner automatisch in Unterordner (Bilder, Dokumente, Musik, Text).
## Was es macht
- Geht durch den Ordner `alifiles`
- Schaut sich die Dateiendung an
- Verschiebt:
  - `.jpg` und `.png` → Ordner `Images`
  - `.pdf` → Ordner `Documents`
  - `.mp3` → Ordner `Music`
  - `.txt` → Ordner `Text`
- Andere Dateien lässt es einfach liegen also unverändert
